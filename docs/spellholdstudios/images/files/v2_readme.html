Overview


This mod changes some inventory icons of Baldur's Gate II to the style of Baldur's Gate. There is a lot of content in Baldur's Gate 2 (especially in the ToB expansion) that simply doesn't live up to the artwork quality the original Baldur's Gate offered--the paperdolls were a good example. Whether it was just rushed work on the developer side or something else, I do not know. This mod aims to replace exactly these parts of the game with content that lives up to the quality we were used to from the original. So, technically speaking, this is an attempt to fix "cosmetic flaws" that should have never made it through QA.


Update as of August 23rd, 2005:

  • It seems there is somewhat of an explanation for the shift, as reading the manual, Bioware's original graphic designer responsible for the art in BG1/ToTSC, Daniel Walker, passed away on June 6th, 1999 of a 'physical disability he had since birth'. In fact, Baldur's Gate II is dedicated to his memory. Let 1pp pay respect to his beautiful and unique style and artwork, and our humble attempt to try and make BGII stay faithful to it.

Compatibility


This Artwork pack is designed to work with Baldur's Gate II: Shadows of Amn with or without the Throne of Bhaal expansion pack. The mod should also work with Tutu and Baldur's Gate Trilogy.

Mod Details


The mod contains inventory item updates, with a split install option for potions. It includes the following:

  • Item graphics matching with the original paperdoll content
  • Some mod item patches (Chloe, BWL's herb add on, etc.)
  • Fixed ToB item graphics (finally in BG1-BG2 style)
  • BG1 ports for some BG2 item graphics (reverting where the original looked better)
  • Various other item fixes, including a "secret" item that was broken, but present, in the original game

By request, the potion inventory BAMs were moved into a second, separate component.
